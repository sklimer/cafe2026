# Админ-панель для системы ресторанов

## Описание

Админ-панель для управления системой ресторанов, позволяющая управлять:
- Ресторанами, филиалами, меню, товарами, категориями, тегами, опциями
- Заказами (просмотр, изменение статуса, отмена)
- Пользователями (клиентами)
- Акциями, промокодами, бонусной системой
- Рассылками
- Аналитикой и отчетностью

## Технологический стек

- **Frontend**: React 18 + TypeScript
- **UI Framework**: Material-UI (MUI)
- **Charts**: Chart.js + react-chartjs-2
- **Routing**: React Router DOM
- **State Management**: React hooks
- **Build Tool**: Vite

## Установка

```bash
npm install
```

## Запуск

```bash
npm run dev
```


## Функциональные возможности

### Дашборд
- Статистика за сегодня: заказы, выручка, новые пользователи
- Графики за неделю/месяц по ключевым метрикам
- Последние заказы с быстрыми действиями
- Предупреждения (мало товаров, проблемы с заказами)
- Быстрые действия (создать заказ, добавить товар и т.д.)

### Управление ресторанами
- Список ресторанов с фильтрами и поиском
- Создание/редактирование ресторанов
- Управление филиалами ресторана

### Управление меню
- Категории: древовидная структура, порядок отображения
- Товары: основная информация, изображения, характеристики
- Теги: создание и управление тегами
- Опции товаров

### Управление заказами
- Список заказов с расширенными фильтрами
- Детали заказа
- Быстрое изменение статуса
- Массовые операции
- Экспорт заказов

### Управление клиентами
- Список пользователей с фильтрами
- Детальная информация о пользователе
- Массовые действия
- Сегментация пользователей

### Акции и промокоды
- Создание и управление акциями
- Создание промокодов
- Отслеживание использований

### Аналитика
- Аналитика в реальном времени
- Графики по периодам
- Топ товаров, категорий, ресторанов
- Отчеты
- Экспорт отчетов

## Особенности реализации

- Адаптивный дизайн
- Темная/светлая тема
- Цветовая кодировка статусов
- Массовые операции
- Real-time обновления (в будущем)
- Горячие клавиши (в будущем)
